    Q&A


    What are the 3 main differences between web 1.0, web 2.0 and web 3.0?

    • Web 1.0 is a read only version.
      The websites contain information that you could search, but have limited interactions and ability between them and the user.
      It basically works just as an information portal to the viewers or clients
    • Web 2.0 is the interactive data writable platform.
      The websites now have the option to to interact with their users or clients. Users can share information (like stack overflow or wikipedia).
      *A good test of whether a site is Web 2.0 site is whether it is made better by adding a user(like facebook).
    • Web 3.0 is the the "semantic web"."
      Semantic web means that now the web can analyse the meaning of words when a user is searching some information, rather than on keywords or numbers.
      It enables the upgradation of back-end of the web, after a long time of focus on the front-end.

    Example of the differences between the webs:

    Let's take for example the use case in which the user is searching for best restaurants in Beer-Sheva
    • in web 1.0 he need to put only keywords and the result that he will get will be based only on the information in the websites that will be relevant for the query.
    • in web 2.0 the user will get the restaurants not only by the information, but he will be able to see the rating that people gave for each restaurant and their recomandations.
    • in web 3.0 the data that exists on that user will help to show him restaurants based on his taste and food that he likes
